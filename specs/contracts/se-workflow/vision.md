# Contract: Software Engineering Worklow

## Overview

Description of the workflow for software engineering tasks provided by SeeAI framework.

## Steps

- Register a change to the project
- Analyze the impact
  - Identify new specs
  - Identify affected specs
  - Stop if specifications need to be created or updated (analysis generates partial output with only specs sections)
- Prepare specs
  - New specs
  - Updated specs
  - Must be completed before proceeding to implementation planning
- Prepare implementation plan
- Execute implementation plan
